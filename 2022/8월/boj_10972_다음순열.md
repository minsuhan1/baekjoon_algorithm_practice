### 코드
```python
n = int(input())
inputArr = list(map(int, input().split(' ')))

def findNext(arr):
    i = len(arr) - 1
    while i > 0:
        if arr[i] > arr[i-1]:
            # arr[i-1] 뒤의 숫자들 중 arr[-1]보다 큰 수들 중 가장 작은 값을 선택하여 자리 교환
            temp = arr[i-1]
            swap = min(list(filter(lambda x : x > temp, arr[i:])))
            arr[arr.index(swap)] = temp
            arr[i-1] = swap
            # arr[i-1] 뒤의 숫자들을 오름차순 정렬한다.
            arr = arr[:i] + sorted(arr[i:])
            return " ".join(list(map(str, arr)))
        else:
            i-=1
    return -1

print(findNext(inputArr))
```

### 풀이 방법
- 순열의 맨 뒤에서부터 arr[i-1] < arr[i]인 두 수를 찾는다. 그러면 arr[i:] 순열은 내림차순 정렬되어있어서 다음 순열이 없다는 뜻이다.
- arr[i-1] 뒤의 순열에서(arr[i:]) arr[-1]보다 큰 수들 중 가장 작은 값을 선택하여 arr[i-1]과 자리를 바꾼 다음, arr[i-1] 뒤의 순열을 `sorted()` 메소드로 오름차순 정렬하였다.
